# Changelog

All notable changes to ASYCUDA Excel Converter are documented here.

## [2.0.0] — 2026-06-22

### Added
- Complete V2.0 rebuild with Next.js 16+ and TypeScript
- Custom design system with CSS variables (no Bootstrap dependency)
- WCAG 2.2 AA accessibility compliance
- Responsive mobile-first design
- Drag-and-drop file upload
- In-memory file processing (no disk writes)
- ASYCUDA-compliant XML generation
- Validation engine with human-readable error messages
- Full documentation suite (7 docs)
- SEO optimization (OG, sitemap, robots.txt, structured data)
- Security headers (CSP, HSTS, X-Frame-Options)
- Performance optimization (90+ Lighthouse target)
- Skip-to-content link
- Semantic HTML landmarks
- Visible focus states
- Touch targets ≥ 44px
- Pricing page with three tiers
- Comprehensive FAQ
- Support and contact pages
- Privacy policy and terms of service
- Project context for AI coding agents
- Sample file download

### Removed
- Bootstrap, jQuery, Font Awesome, Swiper, Owl Carousel, WOW.js, Animate.css
- All legacy plugin dependencies
- Framework soup architecture

### Changed
- Complete visual redesign (dark premium aesthetic)
- Inter font family
- Lucide icon library
- Table-based layout → responsive grid
- Inline color values → CSS variable system

## [1.0.0] — Original Release
- Initial Excel-to-ASYCUDA XML converter
